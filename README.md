### Getting started

#### 1. Step: Fetch `gepard` code

```bash
./scripts/fetch-code.sh
```

#### 2. Step: Install dependencies

```bash
./scripts/install-deps.sh
```

#### 3. Step: Build code and paper

Build `pdf` and `code`.
```bash
cmake -H. -Bbuild
cd build
make
```
